Amministrazione

Diversi sotto-menu per la gestione dei dati possono essere richiamati dal menu principale. Le funzioni per configurare le impostazioni delle applicazioni si trovano nella pagina del menu principale.

Configurazione

La lingua dell’applicazione può essere definita qui. Verrà salvata come cookie presso ogni utente.

Links

Sotto questo titolo si trovano collegamenti a documentazioni e direttive relative alle licenze.

Banca dati

Salvare i dati

Durante il salvataggio i dati saranno inseriti in un archivio SQL sul computer dell’utente. L’utente può scegliere liberamente il nome da dare al file e dove salvarlo.

Immettere la password

Per effettuare una ripresa dei dati salvati, l’utente deve dapprima scegliere un archivio SQL. Se la versione dei dati salvati è compatibile con la versione del programma i dati saranno letti.

Attenzione: Con questa procedura i dati saranno cancellati. Prima di procedere effettuare un salvataggio dei dati esistenti.

Iscrizioni online di ZLV

Registrare i dati

Per registrare i dati online di ZLV, l’utente deve dapprima selezionare il file XML corrispondente. Gli atleti vengono inseriti nei gruppi e squadre corrispondenti.

Svuota la cache

Con questa funzione e possibile eliminare i problemi con gli atleti e/o società non trovati.

Attenzione: Gli atleti e le società selezionate saranno cancellati affinché possano essere nuovamente caricati durante l’aggiornamento della banca dati.
Questo presuppone che per i meeting selezionati in Athletica nessun atleta è registrato.
