Neue Staffel

Um eine neue Staffel zu erfassen, müssen zuerst Kategorie, Staffeldisziplin und Verein ausgewählt werden. Die Disziplinenliste umfasst in diesem Fall nur Staffeldisziplinen.
Staffeln können aber erst erfasst werden, wenn sie auch im Zeitplan eingetragen sind.
Nach erfolgter Auswahl kann ein frei wählbarer Staffelname vergeben werden. Jedoch darf derselbe Name pro Katgorie und Disziplin nur einmal vergeben werden.
Eine Mannschaft kann nur zugeteilt werden, sofern diese Kategorie als Mannschaftswettbewerb (z.B. Vereinsmeisterschaft) durchgeführt wird und die Mannschaft bereits erfasst wurde.
Zusätzlich kann eine Bestleistung erfasst werden, welche bei der Serieneinteilung berücksichtigt werden kann.

Die Athletenliste umfasst all jene Athleten, die für die gewählte Kategorie, den gewählten Verein und, falls vorhanden, für die gewählte Mannschaft in dieser Staffeldisziplin gemeldet sind. Die Athleten sind entweder keiner oder dieser Staffel zugeteilt.
Die Zuteilung eines Athleten zu einer Staffel erfolgt durch die Vergabe einer Staffelpostion (aufsteigend nummeriert). Diese Position kann jederzeit geändert oder gelöscht werden.
Eine Zuteilung von Athleten ist nicht unbedingt erforderlich.
