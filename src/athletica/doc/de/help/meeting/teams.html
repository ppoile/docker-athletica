Mannschaften

Die Liste links zeigt alle angemeldeten Mannschaften. Die Liste kann nach Kategorie oder Mannschaftsname sortiert werden. Durch einen Mausklick auf eine Mannschaft werden deren Details im Hauptteil des Fensters angezeigt.

Befehle

Drucken

Im Hauptteil wird ein Formular zum Drucken von Mannschaftsanmeldungen angezeigt.

Neue Anmeldung

Aufruf einer Eingabemaske zur Erfassung einzelner Anmeldungen.

Mannschaftsdaten ändern

Nach Anklicken einer Mannschaft aus der Liste wird im Hauptteil ein Formular zur Änderung der Mannschaftsdaten geöffnet. Der name ist unabhängig vom Verein und kann dementsprechend frei gewählt werden.

Mannschaft löschen

Um eine Mannschaft zu löschen muss diese zuerst per Mausklick aus der Liste links selektiert werden. Danach kann sie mit dem Schalter Löschen gelöscht werden. Dies ist nur möglich, wenn keine Athleten oder Staffeln zugeteilt worden sind.
