Vereine

Bedienung

Ein neuer Verein wird mit dem Formular am oberen Seitenende aufgesetzt und muss gespeichert werden.
Änderungen an bestehenden Daten werden direkt in der Datenbank gespeichert, wenn zum nächsten Feld gewechselt wird.
Ein Verein kann nur gelöscht werden, wenn er nicht mehr referenziert wird.

Daten

Name
Voller Name des Vereins, z.B LC Brühl.
Sortierwert
Die Vereinsanzeige wird jeweils nach dem hier angegebenen Wert aufsteigend sortiert, z.B Brühl, LC. Es kann auch ein nummerischer Sortierwert eingegeben werden.
