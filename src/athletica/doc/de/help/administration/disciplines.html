Disziplinen

Bedienung

Eine neue Disziplin wird mit dem Formular am oberen Seitenende aufgesetzt und muss gespeichert werden.
Änderungen an bestehenden Daten werden direkt in der Datenbank gespeichert, wenn zum nächsten Feld gewechselt wird.
Eine Disziplin kann nur gelöscht werden, wenn sie nicht mehr referenziert wird.

Für einen bestimmten Typ kann eine pauschale Änderung vorgenommen werden

Daten

Disziplin
Ausführliche Bezeichnung der Disziplin, erscheint z.B. in Titelzeilen.
Kurzname
Abkürzung der Disziplin (max. 10 Zeichen), erscheint in Listen.
Typ
Definiert, wie diese Disziplin vom Programm behandelt wird. Der Typ hat Auswirkungen auf verschiedene Formulare (z.B. Resultateingabe).
Appellzeit
Zeitlimite für den Appell in Minuten vor dem Start; Disziplinen mit einer Appellzeit über Null werden beim Gesamtausdrucken aller Appellblätter berücksichtig. Die Appellzeit wird dabei ausgerechnet.
Staffelgrösse
Anzahl Athleten, welche ein Staffelteam umfasst.
Seriegrösse
Standardwert für die Anzahl Athleten, die gemeinsam in einer Serie starten.
Reihenfolge
Nummerischer Wert, der definiert, in welcher Reihenfolge die Kategorien jeweils angezeigt werden.
Aktiv / Inaktiv
Diszipline können inaktiv gesetzt werden, somit erscheinen sie in keiner Auswahlliste mehr.
