Kategorien

Bedienung

Eine neue Kategorie wird mit dem Formular am oberen Seitenende aufgesetzt und muss gespeichert werden.
Änderungen an bestehenden Daten werden direkt in der Datenbank gespeichert, wenn zum nächsten Feld gewechselt wird.
Eine Kategorie kann nur gelöscht werden, wenn sie nicht mehr referenziert wird.

Daten

Kategorie
Ausführliche Kategoriebezeichnung, erscheint z.B. in Titelzeilen.
Kurzname
Abkürzung der Kategorie (max. 3 Zeichen), erscheint in Listen
Alterslimite
Maximales Athletenalter; wird bei der Anmeldung überprüft.
Reihenfolge
Nummerischer Wert, der definiert, in welcher Reihenfolge die Kategorien jeweils angezeigt werden.
Aktiv / Inaktiv
Kategorien können inaktiv gesetzt werden, somit erscheinen sie in keiner Auswahlliste mehr.
