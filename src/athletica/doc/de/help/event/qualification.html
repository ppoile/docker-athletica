Qualifikation

Der Name der vorhergehenden Runde sowie die Anzahl der qualifizierten Athleten werden angezeigt. Die folgenden Parameter beeinflussen die Serieneinteilung für die folgende Runde:
Anzahl Athleten pro Serie
Wieviele Athleten darf eine Serie maximal umfassen.
Anzahl Bahnen
Ist dieser Wert höher als Anzahl Athleten werden die überzähligen Bahnen leer gesetzt. Dies ist hilfreich, wenn nicht alle Bahnen besetzt werden, die überzähligen Bahnen aber trotzdem angezeigt und auf Formulare ausgedruckt werden sollen.

Dieser Parameter kann nur bei Läufen gesetzt werden.

Einteilungs-Modus
Bei einer Einteilung nach Bestleistung werden die Athleten leistungsmässig pro Serie von den zentralen Bahnen nach aussen verteilt.

Erfolgt die Einteilung nach IWB Regel 166 wird pro Serie die schnellere Hälfte der Athleten den zentralen Bahnen zugelost, die langsamer den äusseren Bahnen.
