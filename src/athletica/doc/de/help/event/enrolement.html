Appell

Die Appellfunktion erlaubt das Ausdrucken von Appellblättern und die Durchführung der Anwesenheitskontrolle.

Drucken

Wird die Druckfunktion ohne Wettkampfauswahl aufgerufen, werden die Appellblätter für all jene Disziplinen gedruckt, die eine Appellzeit grösser als Null haben.
Andernfalls wird das Appellblatt der gewählten Diszipline ungeachtet der Appellzeit ausgedruckt.
Der Status der betroffenen Runden wird auf Appell hängig gesetzt.

Appell abschliessen

Wurden bereits Appellblätter ausgedruckt und steht der Status der ersten Runde dementsprechend auf Appell hängig, wird eine Funktion zum Abschliessen des Appells angezeigt.
Beim Abschliessen wird der Status auf Appell abgeschlossen gesetzt. Erst jetzt können Serien eingeteilt werden.

Liste

In der Liste der für den gewählten Wettkampf angemeldeten Athleten sind alle Athleten als anwesend vorselektiert. Abwesende Athleten können hier gelöscht werden.
Die Liste ist standardmässign nach Namen sortiert, kann aber durch Anklicken der entsprechenden Kolonnenüberschrift auch nach Startnummer oder Verein sortiert werden.
