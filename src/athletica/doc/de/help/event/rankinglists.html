Ranglisten

Mit dieser Funktion können Gesamtranglisten, Ranglisten pro Kategorie, pro Wettkampf, pro Wettkampfrunde oder pro Serie gedruckt werden. Je nach Auswahl stehen verschiedene Druckoptionen zur Verfügung.

Typ

Die folgenden Listen können generiert werden:
Einzel
Einzelranglisten, eventuell eingeschränkt auf eine Kategorie (auch von/bis), einen Wettkampf, Wettkampfrunde, Serien (von/bis) und/oder Disziplinen (von/bis).
Einzel, alle Versuche
Einzelranglisten, eventuell eingeschränkt auf eine Kategorie (auch von/bis), einen Wettkampf, Wettkampfrunde, Serien (von/bis) und/oder Disziplinen (von/bis).
Hier werden zusätzlich die einzelnen Versuchresultate angezeigt
Serien trennen
Für Einzel und für Einzel, alle Versuche können die Resultate bei zusammengefassten Runden wieder getrennt angezeigt werden.
Startnummern
Für Einzel und für Einzel, alle Versuche können die Startnummern angezeigt werden
Gesamtrangliste
Für Einzel und für Einzel, alle Versuche kann eine Gesamtrangliste gewählt werden, die ans Ende der Rangliste angehängt wird.
Mehrkampf
Mehrkampfranglisten, eventuell eingeschränkt auf eine Kategorie (auch von/bis). Pro Athlet werden die Zählresultate pro Disziplin gezeigt.
Mannschaftswettkämpfe
Ranglisten für Mannschaftswettkämpfe, z.B. SVM.
Vereinsrangliste
Rangliste der Teams nach Totalpunktzahl pro Kategorie, eventuell eingeschränkt auf eine Kategorie (auch von/bis). Beim normalen Vereinswettkampft werden pro Team die eingerechneten Disziplinen und je die Anzahl der gewerteten Athleten gezeigt. Bei Vereins-Teammehrkämpfen werden alle gewerteten Mehrkämpfer aufgeführt.
Notenblätter
Formular mit allen Zählresultaten pro Team zur Unterzeichnung durch die Schiedsrichter und Mannschaftsführer. Die Auswahl kann auf eine einzelne Kategorie eingeschränkt werden.

Sortierung

Die Ausgabe der Sortierung nach Ahtletenkategorie bezieht sich nur auf Einzelranglisten.

Seitenumbruch

Je nach Ranglistenauswahl kann nach jeder Kategorie oder nach jeder Disziplin ein Seitenumbruch eingefügt werden.

Optionen

Titelblatt
Ein Titelblatt mit Meetingbezeichnung und Angaben zur Organisation wird vorangehängt.
Bestleistungen
Ist diese Option aktiviert, werden in der Rangliste hinter jedem Athleten die Informationen zu den Bestleistungen (Saison- und persönliche Bestleistung) angezeigt.

Bei vielen Resultaten kann die Ausgabezeit mehrere Minuten dauern

Funktionen

Drucken
Ein neues Browserfenster wird geöffnet und die Druckfunktion gestartet.
Anzeigen
Die Rangliste wird im selben Browserfenster dargestellt.
