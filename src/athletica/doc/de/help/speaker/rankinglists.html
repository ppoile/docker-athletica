Ranglisten

Mit dieser Funktion können Gesamtranglisten, Ranglisten pro Kategorie, pro Wettkampf oder pro Wettkampfrunde angezeigt werden.

Typ

Die folgenden Listen können generiert werden:
Einzel
Einzelranglisten, eventuell eingeschränkt auf eine Kategorie und/oder einen Wettkampf.
Mehrkampf
Mehrkampfranglisten, eventuell eingeschränkt auf eine Kategorie. Pro Athlet werden die Zählresultate pro Disziplin gezeigt.
Mannschaftswettkämpfe
Ranglisten für Mannschaftswettkämpfe, z.B. SVM.
Vereinsrangliste
Rangliste der Teams nach Totalpunktzahl pro Kategorie, eventuell eingeschränkt auf eine Kategorie. Bei normalen Vereinswettkämpfen werden pro Team die eingerechneten Disziplinen und die Anzahl der gewerteten Athleten gezeigt. Bei Vereins-Teammehrkämpfen werden die gewerteten Mehrkämpfer gezeigt.
Notenblätter
Formular mit allen Zählresultaten pro Team zur Unterzeichnung durch die Schiedsrichter und Mannschaftsführer. Die Auswahl kann auf eine einzelne Kategorie eingeschränkt werden.
