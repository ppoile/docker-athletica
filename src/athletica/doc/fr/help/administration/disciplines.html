Disciplines

Emploi

Une nouvelle discipline est ajoutée au moyen du formulaire en haut de la page et doit être enregistrée.
Les modifications apportées aux données existantes sont directement enregistrées dans la base des données lorsqu’on change de case.
Une discipline ne peut être supprimée que si elle n’est plus référencée.

Pour un type précis, il est possible d'effectuer un changement global de l'heure d'appel et de rassemblement

Données

Discipline
Appellation détaillée de la discipline, apparaît par ex. dans les lignes de titre.
Nom abrégé
Abréviation de la discipline (max. 10 car.), apparaît sur les listes.
Type
Définit la manière dont cette discipline est traitée dans le programme. Le type a des effets sur différents formulaires (par ex. introduction des résultats).
Heure d’appel
Temps limite pour l’appel en minutes avant le départ; il est tenu compte des disciplines avec un temps d’appel au-dessus de zéro lors de l’ensemble de l’impression de toutes les feuilles d’appel. Avec cela l’heure d’appel est calculée.
Grandeur des relais
Nombre d’athlètes par équipe de relais.
Grandeur des séries
Valeur standard pour le nombre d’athlètes, qui s’alignent ensemble dans une série.
Ordre
Valeur numérique qui définit l’ordre dans lequel les catégories sont à chaque fois affichées.
Actif / Inactif
Les disciplines peuvent être désactivées, ainsi elles n’apparaissent plus sur aucune liste de sélection.
