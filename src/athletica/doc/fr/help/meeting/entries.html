Inscriptions

La liste à gauche montre tous les athlètes inscrits. Cette liste peut être triée par dossard ou nom. En cliquant avec la souris sur un athlète, ses détails seront affichés dans la partie principale de la fenêtre.

Ordres

Actualiser les meilleures performances

Cette fonction actualise les meilleures performances (plus précisément : performances annoncées pour l’organisation des séries) avec les résultats disponibles dans les données de base. Il est ainsi possible d’actualiser ultérieurement les inscriptions déjà enregistrées (ou arrivées via ajustement du meeting) et leurs performances annoncées. Si l’option "ne pas toucher" est sélectionnée, il ne sera pas écrit par-dessus les meilleures performances entrées manuellement.

Dossards

Dans la partie principale, un formulaire est affiché pour l’attribution des dossards.

Imprimer

Dans la partie principale, un formulaire est affiché pour l’impression Des listes d’inscription.

Nouvelle inscription

Appel d’un masque de saisie pour l’enregistrement de différentes inscriptions.

Rechercher

Dans le champ de recherche on peut soit introduire un nom d’athlète ou un dossard. En pressant sur la touche de saisie la recherche est lancée et la liste des résultats est affichée dans la partie principale. Après avoir cliqué sur un athlète de cette liste, ses données spécifiques sont affichées.

Modifier une inscription

Après avoir cliqué sur une inscription de la liste de gauche ou de la liste de résultats suite à une recherche, un formulaire pour modifier les données de l’athlète est ouvert dans la partie principale. À part société toutes les valeurs peuvent être modifiées (la société ne peut être modifiée que dans la banque de données des athlètes).
Lors du changement de dossard, il est contrôlé que celui-ci ne soit pas déjà attribué. La catégorie ne peut être remise que lorsque toutes les inscriptions aux disciplines ont été supprimées. Après avoir quitté le champ de saisie correspondant, toutes les modifications sont immédiatement inscrites dans la banque des données.

Une équipe ne peut être répartie que si cette catégorie est organisée comme concours par équipe (par ex. championnats de société) et si l’ équipe a déjà été enregistrée.
Une discipline ne peut être changée que si l’athlète n’a pas encore été réparti dans une série de compétition de la discipline correspondante. Si une discipline est marquée, un nouveau champ formulaire apparaît pour entrer une meilleure performance.

Si l’athlète d’une discipline (surtout relais) a été réparti dans une autre catégorie, ce sera affiché spécialement. Cette inscription de discipline peut également être supprimée.

Supprimer une inscription

Après avoir cliqué sur une inscription, celle-ci peut aussi être supprimée. Ceci n’est possible que si l’athlète n’a pas encore été réparti dans une série de la compétition. Les données générales des athlètes restent toutefois dans la banque de données des athlètes .
